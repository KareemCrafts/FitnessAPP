# Fitness Tracker App

## Overview
Android fitness tracking application for logging workouts, tracking progress, and achieving fitness goals.

## Features
- Workout logging (exercises, sets, reps, weight)
- Progress tracking and workout history
- Goal setting and monitoring
- Exercise library database
- User profile with statistics

## Technologies
- **Platform:** Android
- **Language:** Java/Kotlin
- **Database:** SQLite/Room
- **UI:** XML, Material Design
- **Architecture:** MVVM

## Skills Demonstrated
- Android app development
- SQLite database management
- UI/UX design with Material Design
- Data persistence
- Activity/Fragment lifecycle management

## Future Enhancements
- Exercise timer
- Workout plans
- Calorie tracking
- Cloud sync
- Social features
